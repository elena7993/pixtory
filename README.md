# Pixtory

### 픽셀(레트로)이미지 변환 앱 👀

### 1.목적

- 사용자가 원하는 이미지를 레트로아트로 만들 수 있다.
- 나만의 독특한 아이콘, 로고 등을 만들 수 있다.

### 2.기능

- 이미지 업로드와 다운로드
- 이미지 변환
- 변환된 이미지를 공유

### 3. 스택

- HTML/CSS, JS, React, Canvas
